## 二、身价计算公式

```
市场身价 = base × abilityMult × ageFactor × potBonus
```

### 参数说明

**base（基准值）**
```
base = 1,500,000（人民币）
```

**abilityMult（能力系数）**
```
normalized = (CA - 100) / 80
abilityMult = 10 ^ (max(0, normalized) × 3.2)
```
能力低于100时 abilityMult = 1，高于100后指数增长。

**ageFactor（年龄系数）**

| 年龄段 | 计算方式 |
|--------|----------|
| 16 – 18 | 0.45 + (age - 16) × 0.08 |
| 19 – 22 | 0.60 + (age - 18) × 0.07 |
| 23 – 27 | 0.88 + (age - 22) × 0.022 |
| 28 – 30 | 0.99 - (age - 27) × 0.07 |
| 31 – 32 | 0.78 - (age - 30) × 0.10 |
| 33 – 34 | 0.58 - (age - 32) × 0.16（悬崖加速） |
| 35 | 0.26 - (age - 34) × 0.20（谷底） |

最低值钳制：`ageFactor = max(0.05, ageFactor)`

**potBonus（潜能溢价）**

仅当 PA > CA 时生效，且随年龄递减：

```
gap = max(0, PA - CA)

if age <= 22: potBonus = 1 + gap × 0.007
if age <= 26: potBonus = 1 + gap × 0.003
if age >  26: potBonus = 1（无溢价）
```

**最终值**
```
身价 = max(50,000, round(base × abilityMult × ageFactor × potBonus))
```

## 三、薪资计算公式

```
周薪 = max(500, round(身价 / 2500))
年薪 = 周薪 × 52
```

薪身比（年薪 / 身价）通常在 2% – 4% 之间，高龄球员因身价下跌薪身比会显著偏高，这是游戏设计刻意制造的决策压力。

## 四、俱乐部收入

本游戏**只有两级联赛**，收入来源为转播费和赞助收入，**不含门票和球场相关收入**。

| 联赛 | 转播收入 | 赞助收入 | 总营收 |
|------|----------|----------|--------|
| 顶级联赛 | ¥6.48 亿 | ¥2.16 亿 | ¥8.64 亿 |
| 次级联赛 | ¥4.02 亿 | ¥1.34 亿 | ¥5.36 亿 |

次级联赛总营收约为顶级联赛的 62%，降级会造成显著的收入落差，是游戏财务压力的核心来源之一。

## 五、财务健康度（薪资压力）

参考 FFP（财政公平竞争）规则：

```
薪资健康度 = 全队总薪资 / 总营收 × 100%
```

| 健康度 | 状态 | 建议处理 |
|--------|------|----------|
| < 60%  | 财务健康 | 可继续引援 |
| 60% – 79% | 需要注意 | 控制引援预算 |
| ≥ 80% | 超支风险 | 触发财务警告，限制转会 |

### 估算全队薪资的简化公式

以阵容中薪水最高的主力球员（核心薪水）为基准：

```
全队总薪资 ≈ 核心薪水 × 3          （3名核心主力）
           + 核心薪水 × 0.45 × 14  （14名主力，45%薪水）
           + 核心薪水 × 0.22 × 8   （8名边缘球员，22%薪水）
```

## 六、35岁退役机制

### 三级预警系统

| 年龄 | 预警级别 | 提示内容 |
|------|----------|----------|
| 30 – 32 | 灰色提示 | 「距离退役还有 N 年，续约年限需谨慎」 |
| 33 – 34 | 橙色警告 | 「处于身价下行通道，建议尽快出售或规划接班」 |
| 35 | 红色警报 | 「本赛季结束后强制退役，续约无效，身价清零」 |

### 设计意图

- 33岁起 ageFactor 进入"悬崖加速"区，身价曲线肉眼可见地折断
- 35岁球员薪身比极高（高薪低值），应对玩家形成自然的出售压力
- 潜能值对30岁以上球员完全无效，消除老将的"潜力泡沫"
- 续约系统中，35岁球员签任何年限合同均在赛季末自动作废
