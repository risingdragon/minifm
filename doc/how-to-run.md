﻿﻿﻿# 如何运行这个游戏

## 环境要求

- Node.js (推荐使用最新稳定版本)
- npm 或 yarn

## 安装依赖

在项目根目录下执行：

```bash
npm install
```

## 启动开发服务器

运行以下命令启动开发服务器：

```bash
npm run dev
```

启动后，打开浏览器访问显示的本地地址（通常是 `http://localhost:5173`）即可开始游戏。

## 构建生产版本

如果需要构建生产版本，执行：

```bash
npm run build
```

构建完成后，产物会生成在 `dist` 目录中。

## 预览生产版本

构建完成后，可以预览生产版本：

```bash
npm run preview
```

## 项目说明

这是一个基于 Vite + React + TypeScript 开发的足球经理游戏。游戏包含以下核心功能：

- 球队阵容管理
- 比赛日程安排
- 比赛模拟
- 积分榜排名

项目结构：
- `src/` - 源代码目录
- `assets/` - 静态资源
- `spec/` - 项目规格文档
