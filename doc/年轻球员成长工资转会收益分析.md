# 年轻球员成长、工资和转会收益分析

## 结论

如果只看“未来转会费增加”能否覆盖“期间工资消耗”，当前数值下大多数年轻球员不值得单纯为了转卖而长期留队。

比较容易接近打平的只有一种情况：低能力、低工资、高潜力、年龄 17-18 岁的球员，并且最好在赛季末年龄增长和合同年限减少之前出售。能力稍高后，工资增长很快，转会费增量通常覆盖不了工资。

如果球员能进入轮换或首发，成长带来的竞技收益要另算；本文只评估“养成后卖钱”这件事。

## 当前公式口径

### 成长

每场比赛后，30 岁及以下球员有机会能力 +1：

```text
成长概率 = 0.06 * 年龄系数 * 潜力差系数 * 低能力加成 * 随机波动
```

关键系数：

| 条件 | 系数 |
|---|---:|
| 年龄 <= 18 | 2.5 |
| 年龄 19-21 | 1.8 |
| 年龄 22-24 | 1.2 |
| 年龄 25-27 | 0.6 |
| 年龄 28-30 | 0.2 |
| 能力 <= 30 | 2.0 |
| 能力 <= 60 | 1.6 |
| 能力 <= 90 | 1.3 |

潜力差越大，成长越快；但单场成长概率最高被限制在 30%。

### 工资

```text
周薪 = 0.1 * (max(能力, 10) / 10) ^ 2.2 * 随机系数(0.8-1.2)
```

- 单位是“万 / 周”。
- 每赛季 38 轮，每轮扣 1 次整队周薪，所以单个球员赛季工资约为 `周薪 * 38`。
- 成长后当前代码只重算身价，不会立刻按新能力涨工资；工资主要在合同到期续约时重新签。

### 转会费

球员身价：

```text
身价 = 能力^2 * 0.04 * 年龄系数 * 潜力系数 * 合同系数
```

出售价格：

```text
出售价格 = floor(身价 * 年龄折扣)
```

年龄折扣：

| 年龄 | 折扣 |
|---|---:|
| <= 24 | 0.85 |
| 25-27 | 0.80 |
| 28-29 | 0.75 |
| >= 30 | 0.70 |

很重要的一点：合同剩余年限会影响身价。每过一个赛季，年龄 +1、合同年限 -1，所以即使能力涨了，赛季末出售价格也可能因为合同变短、年龄变大而下降。

## 模拟口径

模拟采用当前代码公式，按代表性球员跑 20,000 次：

- 每赛季 38 次赛后成长判定。
- 工资按期初能力的期望工资估算。
- 年轻球员初始合同按常见满合同估算：20 岁及以下 5 年，21-27 岁 4 年。
- “赛季末出售”包含年龄 +1、合同 -1 后的价格。
- “赛季结束前出售”只计算 38 场成长，不扣年龄和合同，代表更理想的卖点。

## 赛季结束前出售

这个口径对养成最有利，因为还没吃到年龄和合同缩水。

| 球员类型 | 初始年龄/能力/潜力 | 平均能力增长 | 转会费增量 | 工资消耗 | 净收益 | 盈利概率 |
|---|---|---:|---:|---:|---:|---:|
| 第6级低能高潜 | 17 / 20 / 50 | +6.1 | +20.1 万 | 19.0 万 | +1.1 万 | 60.6% |
| 第6级较好苗子 | 18 / 35 / 60 | +4.2 | +19.7 万 | 60.8 万 | -41.1 万 | 0% |
| 第5级可用新秀 | 19 / 50 / 80 | +3.7 | +22.3 万 | 129.2 万 | -106.9 万 | 0% |
| 中档潜力股 | 21 / 70 / 100 | +3.1 | +18.1 万 | 273.6 万 | -255.5 万 | 0% |
| 高能力潜力股 | 22 / 100 / 130 | +1.8 | +12.2 万 | 600.4 万 | -588.2 万 | 0% |
| 接近满潜 | 18 / 45 / 50 | +1.8 | +7.8 万 | 102.6 万 | -94.8 万 | 0% |

短线结论：只有“17 岁、能力 20、潜力 50”这类极低工资球员，靠成长涨价有机会覆盖工资，而且收益很薄。能力到 35 以后，工资已经明显超过转会费增量。

## 赛季末出售

这个口径更接近正常赛季结算后再卖。因为年龄增长和合同减少会压低身价，结果更差。

| 球员类型 | 持有时间 | 平均能力增长 | 转会费增量 | 工资消耗 | 净收益 |
|---|---:|---:|---:|---:|---:|
| 第6级低能高潜 17/20/50 | 1 年 | +6.1 | +13.1 万 | 19.0 万 | -5.9 万 |
| 第6级低能高潜 17/20/50 | 2 年 | +10.8 | +18.9 万 | 38.0 万 | -19.1 万 |
| 第6级低能高潜 17/20/50 | 3 年 | +13.3 | +15.5 万 | 57.0 万 | -41.5 万 |
| 第6级较好苗子 18/35/60 | 1 年 | +4.2 | +4.8 万 | 60.8 万 | -56.0 万 |
| 第5级可用新秀 19/50/80 | 1 年 | +3.7 | -7.3 万 | 129.2 万 | -136.5 万 |
| 中档潜力股 21/70/100 | 1 年 | +3.0 | -30.0 万 | 273.6 万 | -303.6 万 |

长期结论：赛季末再卖时，低能力高潜球员也很难赚钱；能力 50 以上的年轻球员即使成长不错，工资和合同缩水通常会把转卖收益吃掉。

## 实用建议

1. 纯转卖套利只适合极低工资球员

   优先看 17-18 岁、能力 <= 25、潜力至少比能力高 25 点以上、工资 0.1-0.3 万 / 周的球员。即便如此，也更像低风险彩票，不是稳定赚钱方式。

2. 能力 35 以上不要只为卖钱而囤

   这类球员工资已经开始明显上升，单赛季工资往往超过成长带来的转会费增量。除非他能提供比赛贡献，否则纯投资回报偏差。

3. 最佳卖点通常不是赛季末

   如果只是为了卖钱，赛季结束前卖更好。赛季末年龄 +1、合同 -1 后，身价可能被压低。

4. 合同年限很关键

   当前身价公式里合同系数很重。球员成长了但合同变短，出售价格可能不涨反跌。未来如果增加续约按钮，低薪长约的新秀会更有投资价值。

5. 高能力年轻球员应该按竞技资产看

   能力 50、70、100 这类球员，工资消耗远高于转会费增量。留他们的理由应该是提升战绩、升级收入、或替代更贵球员，而不是期待卖出盈利。

## 总判断

当前版本的养成经济并不是“低买年轻人，养几年高卖”的模型，而更像“年轻球员低薪自用，顺便成长”。  

如果目标是赚钱，建议只保留非常便宜的高潜小妖，并在成长后尽早出售；如果目标是升级，年轻球员成长的主要价值在于提高球队实力，而不是转会费差价。
