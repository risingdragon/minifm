# 球员成长系统 - 验证检查清单

- [ ] Checkpoint 1: Player 类型包含 potential 字段
- [ ] Checkpoint 2: 新生成的球员包含有效的 potential 值（0-200）
- [ ] Checkpoint 3: 年轻球员（<23岁）在比赛结束后 overall 增加
- [ ] Checkpoint 4: 黄金年龄球员（23-29岁）在比赛结束后 overall 增加但幅度较小
- [ ] Checkpoint 5: 高龄球员（≥30岁）在比赛结束后 overall 下降
- [ ] Checkpoint 6: 赛季结束后球员年龄增加1岁
- [ ] Checkpoint 7: 球员成长逻辑在比赛结束时正确执行
